# 🗺️ DSA Visualizer: The Quest for Mastery
### *A Gamified Learning Environment for Data Structures*

[![Live Demo](https://img.shields.io/badge/Demo-Live%20Now-brightgreen?style=for-the-badge&logo=google-chrome)](https://AnAnyA-X.github.io/DSA-Visualizer/dsaquest/index.html)

## 🌟 Overview
**DSA Visualizer** is an interactive, web-based platform designed to bridge the gap between theoretical data structures and practical understanding. By combining a **gamified quest dashboard** with high-fidelity visualizers, it provides an intuitive way for students to grasp complex algorithms.

Selected by the **DSA Department** as a recommended tool for peer-learning and senior-year demonstrations.

---

## 🚀 Key Modules

### 🏯 1. The Quest Dashboard (`/dsaquest`)
An immersive UI that acts as a central hub. It uses a "Map-style" navigation to guide users through different challenges, making the learning process engaging rather than repetitive.

### 🌳 2. Professional B+ Tree Visualizer (`/bplustree`)
A robust implementation of the B+ Tree data structure.
- **Visual Splits & Merges:** Real-time animation of node overflows and underflows.
- **Parameter Control:** Users can define the 'Order' (m) of the tree to see structural changes.
- **Algorithm Transparency:** Displays the step-by-step path from Root to Leaf.

### 🔝 3. Max Heap Visualizer (`/Heap`)
Designed to show the duality of Heaps (Array vs. Tree).
- **Interactive Insertion:** Watch the "Bubble Up" (Heapify-up) logic.
- **Dynamic Deletion:** Visualizes the "Sift Down" (Heapify-down) process.
- **Array Mapping:** Highlights exactly which index in the array corresponds to which node in the tree.

## 🛠️ Technical Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+).
- **Architecture:** Modular Component Design – Each visualizer is independent yet connected via the main hub.
- **Deployment:** GitHub Pages for high-availability access.

---

## 📖 Instructions for Use
1. **Explore the Map:** Start at the `dsaquest` dashboard.
2. **Interact:** Click on any Data Structure card to enter its specialized visualizer.
3. **Experiment:** Use the control panels to insert, delete, or search for custom values.
4. **Return:** Use the integrated "Back to Dashboard" navigation to switch topics.
