#  DSA Visualizer Suite
An interactive web-based tool to visualize complex data structures.

### 🛠️ Structures Included:
1. **Max-Heap**: Visualizes priority queue logic with "Bubble Up/Down" and Undo functionality.
2. **B+ Tree**: Demonstrates database indexing and multi-level branching.
3. **Red-Black Tree**: Shows self-balancing binary search tree properties.
